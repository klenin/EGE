  Генератор logic_var_set
   Источник
    Демонстрационные варианты ЕГЭ по информатике 2012, официальный
    информационный портал ЕГЭ. Задание B15.

   Описание
    Решение:

    *   Для начала рассмотрим случай 4х переменных:

            ((x1 ≡ x2) \/ (x3 ≡ x4)) /\ (¬(x1 ≡ x2) \/ ¬(x3 ≡ x4)) = 1

        Уравнение выше можно записать так:

            x1 = x2 <=> x3 != x4
            x1 != x2 <=> x3 = x4

        8 наборов переменных удовлетворяют этому уравнению.

    *   Теперь переходим к случаю 6ти перменых: Для каждого из уже выбранных
        наборов для случая 4х переменных можем добавить по 2 набора из x5,
        x6

    *   Для n = 2*k получаем:

        Всего k-1 пар. Первая пара даёт 8 вариантов, каждая последующая
        увеличивает число вариантов вдвое. Имеем:

            8 * 2 ^ ( k - 2 ) = 2 ^ ( k + 1)
